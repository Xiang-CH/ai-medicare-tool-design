# MediAssist

面向 **COMP7505 — 用户界面设计与开发** 课程的 **AI 辅助医疗文书 / 临床决策支持** 设计原型项目（**Track A：智能自适应界面**）。侧重交互设计与高保真原型（如 Figma），**不要求**实现生产级代码。

项目重点：面向临床医生病历文案工作自动化系统，同时可以增加患者端和审计端。

## 仓库结构

| 路径 | 说明 |
|------|------|
| `project-brief-track-a.md` | 项目说明（英文） |
| `project-brief-track-a.zh.md` | 项目说明（中文） |
| `persona/` | 用户画像（医生、患者、审计方等） |
| `featureCard/` | 功能与流程说明，供原型对照 |
| `AGENTS.md` | 协作者与 AI 助手的设计协作约定 |

## 快速上手

1. 阅读上述项目说明中的 Track 要求与交付清单。
2. 做界面与流程时对照 `persona/` 与 `featureCard/`。
